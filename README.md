Description
-----------

DEXTRE: decorrelation stretch for a multispectral image.

Release: 1.0, September 2025.

Programming language: Matlab 24.1 (R2024a).

License: open source.

This Matlab package implements the algorithms described in the paper:

E. Crabu, F. Pes, and G. Rodriguez.
Numerical methods for decorrelation stretch.
Mathematics, 13:3297 (16 pages), 2025.

---

Installation
------------

After downloading the package, to use it, the user can add the routines directly into the working directory or can add dextre to the Matlab search path by the "addpath" command.

---

PACKAGE USE
-----------

See the cited paper and testdextre.m

---

Package structure
-----------------

The package contains the following files:

* README.md    : this file
* dextre.m     : decorrelation stretch for a multispectral image
* mgs.m        : an implementation of the modified Gram-Schmidt process
* pimentel.ppm : a test dataset
* testdextre.m : test script for dextre.m
